Как это работает
1. Загрузка Excel Используется лист «Источник» с недельным расписанием обучения.
2. Обработка данных Берутся только курсы Нижнекамска, аудитории и преподаватели.
3. Генерация презентации На выходе формируется готовый файл .pptx для скачивания.

Сформировать презентацию

Загрузите Excel-график. Приложение автоматически сформирует презентацию с шаблонным оформлением, датами, аудиториями и курсами на каждый день.

Внутренний инструмент
Поддерживаются файлы .xlsx, .xls, .xlsm. Используется недельный график обучения с листа Источник.
Ожидаю Excel-файл.
Фильтрация В презентацию попадают только курсы Нижнекамска.
Сортировка Порядок на слайде — по аудиториям, полигоны и пустые в конце.
Результат Готовая PowerPoint-презентация для скачивания одним файлом.

Что учитывается

Период курса Курс попадает на каждый день недели, если он реально идёт в этот день.
Преподаватели Используется первый преподаватель из ячейки, в формате «Имя Фамилия».
Локации Аудитории и этажи берутся из места проведения. «Полигоны» выводятся отдельно.
Формат выгрузки Слайды создаются в PowerPoint с оформлением, близким к корпоративному шаблону.
Перед генерацией убедитесь, что в файле корректно заполнены поля: Место проведения, Название мероприятия, Дата начала, Дата завершения, КТ/сопровождающий.
